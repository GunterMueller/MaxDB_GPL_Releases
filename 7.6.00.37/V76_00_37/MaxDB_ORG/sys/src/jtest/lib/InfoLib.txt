This directory can be used to store additional jar - files for the java tests.
All these jar's are added automatically to the classpath.

                  LGPL
mail.jar        - JavaMail API, http://java.sun.com/products/javamail/,
                  SUN specific (binary distribution allowed)
patregex.jar 	- Regular Expressions in Java, http://www.javaregex.com/,
                  LGPL
xalan.jar       - XSLT, http://xml.apache.org/xalan-j/index.htm,

                  Apache Licence
