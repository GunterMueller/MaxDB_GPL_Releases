Index Engine Monitoring

Here you can monitor your registered Index Engines.
You can obtain the following information:
  • Which Index Engines are registered?
  • On which server nodes do these registered Index Engines run?
  • Which Indexing Service Definition do they use?
  • When did they last communicate?

Use the Overview link to get an overview of all your registered Index Engines.

Choose the name of a certain Index Engine from the menu to obtain more detailed information on this particular Index Engine.
